#ssrpanel
基于宝塔面板一键安装ssrpanel脚本。
----------------------------------------------------------------------------
前提条件：

1.安装宝塔面面板，PHP版本选择7.1，其他默认，使用Nginx环境。

2.新建站点，记住域名。

3.新建数据库，记住数据库用户和密码，（数据库名同用户名）。

4.运行下面的脚本：


    wget https://coding.net/u/cvc/p/ssrpanel/git/raw/master/install.sh && bash install.sh


----------------------------------------------------------------------------
根据要求进行操作即可。

主要需要输入的内容如下：


请输入宝塔面板添加的网站域名：(请不要修改添加之后的默认地址，只输入域名即可)

请输入宝塔面板添加的MySQL用户名：

请输入宝塔面板添加的MySQL密码：

请等待系统自动操作......

已加载插件：fastestmirror
